# Recommended Screenshots

This folder contains the recommended visual assets to include in the documentation. 

## 1. dashboard-overview.png
* **Where it should be used:** `docs/README.md`
* **What should be highlighted:** The main analytics chart to show users what a successful account looks like.

## 2. verify-domain.png
* **Where it should be used:** `docs/getting-started/verify-domain.md`
* **What should be highlighted:** Add a red box around the green "Verified" badge next to the SPF and DKIM records.

## 3. csv-import.png
* **Where it should be used:** `docs/troubleshooting/csv-import.md`
* **What should be highlighted:** Highlight the dropdown menu where the user maps the "Email" column from their file to the "Email" field in ShrFlow.

## 4. campaign-builder.png
* **Where it should be used:** `docs/getting-started/first-campaign.md`
* **What should be highlighted:** Point an arrow to the "Send Test Email" button in the top menu.

## 5. analytics-dashboard.png
* **Where it should be used:** `docs/troubleshooting/common-issues.md`
* **What should be highlighted:** Box around the "Opens" and "Clicks" percentages.
